# Rasin Sellers

An upscale, luxury e-commerce experience built using **React** and **Vite**.

## Features

- **Sleek Aesthetic**: Minimalist, editorially styled dark theme utilizing *Playfair Display* and *Inter* fonts.
- **The Curated Collection**: A bespoke catalog of premium items with high-resolution imagery.
- **Localized Pricing**: Prices dynamically calculated and displayed in Nepali Rupees (NPR).
- **Interactive Shopping Bag**: Slide-out cart drawer with real-time quantity adjustments and item removals.
- **Bespoke Checkout Flow**: Captures recipient details, delivery location in Nepal, and mock online payment preferences (eSewa, Khalti, or Direct Bank Wire).
- **Private Orders Archive**: A dedicated page displaying full order tracking details, payment methods, and curating status.

## Technologies Used

- **Framework**: React 19
- **Bundler**: Vite 8
- **Styling**: Custom Vanilla CSS

## Getting Started

### Prerequisites
Make sure you have **Node.js** and **npm** installed on your system.

### Installation
1. Install the required dependencies:
   ```bash
   npm install
   ```

2. Start the local development server:
   ```bash
   npm run dev
   ```

3. Open the link provided in the terminal (e.g. `http://localhost:5173/`) in your browser.
